# LeNet-5 Implementation

This project contains an implementation of the classic LeNet-5 convolutional neural network architecture for image classification tasks, such as digit recognition on the MNIST dataset.

## Features
- Clean and modular code structure
- Training and evaluation scripts
- Easily customizable for other datasets

## Usage
1. Clone the repository.
2. Install dependencies (see requirements.txt).
3. Run the training script to train LeNet-5 on your dataset.
